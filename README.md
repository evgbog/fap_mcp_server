# MCP-сервер для Платформы Форсайт

## ✨ Возможности

- Создание ИИ-агентов с возможностью выполнения запросов к многомерным данным на естественном языке
- Гибкый выбор (подключение) многомерных источников (кубов) для ИИ-анализа
- Возможность переопределить метаданные источника (наименования/описание источников, измерений, мер, форматы данных, единицы измерения и др.)
- Автоматическая вставка результата в выбранный визуализатор Инфопанели
- Генерация и копирование в буфер обмена HTML-кода внешнего плагина Инфопанели (удобно для создания примеров)

## 🚀 Установка

### Расположение компонентов

1. **`python/`** — реализация MCP-сервера на языке Python с использованием библиотеки FastMCP.
2. **`fap_component/`** — компонент для Платформы Форсайт (обработка OLAP-запросов, мета-описания, интерфейс настройки).
3. **`examples/`** — примеры, включая системный промпт агента **«BI аналитик»**.

### Установка компонента в репозиторий Платформы

Установите обновление `fap_mcp_server_1.x.x.pefx` из папки `fap_component`. В процессе установки в папке компонентов будет создана папка `ИИ. MCP сервер ФАП`, в которой будет установлена БД `DB_MCP_META`, указывающая на БД метаданных текущего репозитория, созданы 3 таблицы для хранения метаданных, и ряд дополнителных объектов.

Ключевые объекты:
- Сервисный модуль для MCP-сервера (`MOD_MCP_SERVER`)
- Веб-форма Редактор источников для ИИ-анализа (`FRM_MCP_SETUP`)

### Установка MCP-сервера на языке Python

Реализация сервера находится в файле `fap_mcp_server.py`. Предполагаем, что сервер будет развернут на Linux-сервере в режиме streamable-http. Создайте виртуальную среду, установите необходимые зависимости, при необходимости оформите запуск в виде службы. По умолчанию используется порт 8000, измените при необходимости.

Дополнительно в папке имеется реализация MCP-сервера с встроенной генерацией синтетических данных - `test_mcp3.py`. Мы использовали его для тестирования, он не требует наличия Форсайт-части.

## 🛠 Настройка

### Генерация токена для выполнения кода на стороне Платформы

Реализация MCP-сервера на Python использует API Платформы для выполнения Fore-кода `ForeExecAtomic`:
https://help.fsight.ru/ru/mergedProjects/ppsomservice/operations/repository/foreexecatomic.htm

Для работы этой функции необходимо в `Менеджере безопасности` создать технического пользователя с правами, позволяющими читать метаданные компонента и данные многомерных кубов, после чего сгенерировать соответсвующий токен. Порядок генерации токена описан по ссылке выше.

### Настройка Python-скрипта, реализующего сервер MCP

В коде MCP-сервера задайте значение 4 строковых параметров:

`SERVICE_URL` - сервисный URL (тот же, что указан в параметре `serviceUrl` файла `config.json` веб-приложения)
`FORE_EXEC_TOKEN`  - токен, сгенерированный для выполнения Fore-кода
`METABASE_ID` - идентификатор метабазы
`MCP_MODULE_KEY` - ключ сервисного модуля `MOD_MCP_SERVER` (числовое значение в виде строки). Значение ключа можно посмотреть в окне "Свойства" данного модуля.
