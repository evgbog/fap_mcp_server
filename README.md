# MCP-сервер для Платорфмы Форсайт

Монопо-репозиторий, содержащий несколько компонентов.

## Структура проекта

- **`python/`** — Python-скрипты и backend-утилиты
- **`foresight-scripts/`** — Скрипты на Fore, объекты репозиотрия Форсайт
- **`docs/`** — Документация
- **`examples/`** — Готовые примеры
