# MCP-сервер для Платорфмы Форсайт

## ✨ Возможности

- Создание ИИ-агентов с возможностью выполнения запросы к многомерным данным на естественном языке
- Гибкый выбор (подключение) многомерных источников (кубов) для ИИ-анализа
- Возможность переопределить метаданные источника (наименования/описание источников, измерений, мер, форматы данных, единицы измерения и др.)
- Автоматическая вставка результата в выбранный визуализатор Инфопанели
- Генерация и копирование в буфер обмена HTML-кода внешнего плагина Инфопанели (удобно для создания примеров)

## 🚀 Установка

### Основные компоненты

1. **`python/`** — реализация MCP-сервера.
2. **`fap_component/`** — компонент для Платформы Форсайт (обработка OLAP-запросов, мета-описания, интерфейс настройки).
3. **`examples/`** — примеры, включая системный промпт агента **«BI аналитик»**.
